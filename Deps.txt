Howto use it

Debian :
sudo apt install python-gi python-gi-cairo python3-gi python3-gi-cairo gir1.2-gtk-3.0 youtube-dl 
Install ffmpeg . Or put the static buil on /usr/bin/ (https://johnvansickle.com/ffmpeg/)

Ubuntu:
sudo apt install python-gi python-gi-cairo python3-gi python3-gi-cairo gir1.2-gtk-3.0 youtube-dl ffmpeg

Msys2 :
pacman -S mingw-w64-x86_64-gtk3 mingw-w64-x86_64-python2-gobject mingw-w64-x86_64-python3-gobject
pip3 install youtube_dl
Download ffmpeg.exe (static build) and put it on vd.py folder

Arch Linux :
sudo pacman -S python-gobject python2-gobject gtk3 youtube-dl ffmpeg

Open Suse :
sudo zypper install python-gobject python3-gobject gtk3 youtube-dl ffmpeg

Fedora:
sudo dnf install pygobject3 python3-gobject gtk3 youtube-dl 
Install ffmpeg . Or put the static buil on /usr/bin/ (https://johnvansickle.com/ffmpeg/)
